# Tower Flood Race 커뮤니티 영상 리스트
#
# 사용법
#   - 한 줄에 하나의 URL (YouTube / TikTok)
#   - 줄 시작이 # 이면 주석 (무시됨)
#   - 빈 줄 무시
#   - 위에서부터 표시 순서
#   - 저장 후 페이지 새로고침만 하면 즉시 반영
#
# 섹션 마커 (롱폼/숏폼 분류)
#   [long]   ← 이 줄 아래의 URL 들은 모두 "롱폼" 으로 분류
#   [short]  ← 이 줄 아래의 URL 들은 모두 "숏폼" 으로 분류
#   마커 미지정 영역의 URL 은 패턴으로 자동 분류:
#     - youtube.com/shorts/...  → 숏폼
#     - tiktok.com/...          → 숏폼
#     - youtu.be / watch?v=...  → 롱폼
#
# 메타데이터 (제목·작성자·썸네일) 은 oEmbed 로 자동 채움 (실패 시 기본값 표시)
#
# 예시
#   [long]
#   https://www.youtube.com/watch?v=abc123
#   https://youtu.be/def456
#
#   [short]
#   https://www.youtube.com/shorts/xyz
#   https://www.tiktok.com/@user/video/7891234567890
